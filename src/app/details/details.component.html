Certificate Type

Sign DSC can only be used for signing documents. The most popular usage of is signing the PDF file for Tax Returns, MCA and other websites.

Encrypt DSC can only be used to encrypt a document, it is mainly used in the tender portal. Encrypt DSC is fit for e-commerce documents, legal documentation and sharing documents that are highly confidential and contains information that needs to be protected.

Sign & Encrypt DSC can be used for both signing and encrypting. It is convenient for users who need to authenticate and maintain the confidentiality of the information shared. Its usage includes filing government form and application.

Certificate Class

Class 2

It is used in various form-filling, online registration, email attestation, income tax filing, LLP registration, IE code registration, Form 16,etc

Class 3

it is more secure in comparison to class 2 certificate, This is the safest of all certificates. It is used in E-tendering, Patent and trademark e-filing, Customs e-filing, E-procurement, E-biding E-auction,etc
